## ⚠️ 常见问题及解决方案

### 问题 1：显示"Python 未安装"
**解决方案：**
1. 下载 Python：https://www.python.org/downloads/
2. 安装时**务必勾选** "Add Python to PATH"
3. 重启命令行窗口
4. 运行 `python --version` 验证安装

### 问题 2：显示"找不到文件"
**解决方案：**
1. 确认文件在正确位置：
   ```
   C:\Users\DELL\Desktop\AI编程类文件夹\holland_test_preview.html
   ```
2. 检查文件名是否正确（注意下划线 `_`，不是空格）
3. 确认文件没有被移动或删除

### 问题 3：显示"端口被占用"
**解决方案：**
1. 脚本会自动尝试其他端口，通常不需要手动处理
2. 如果想手动释放端口：
   ```bash
   # 查找占用8000端口的进程
   netstat -ano | findstr :8000
   # 结束进程（替换PID为实际进程ID）
   taskkill /PID <进程ID> /F
   ```

### 问题 4：浏览器显示空白页面
**解决方案：**
1. 按 `Ctrl + F5` 强制刷新
2. 检查浏览器控制台（F12）是否有错误
3. 尝试使用不同的浏览器
4. 清除浏览器缓存

### 问题 5：双击文件没有反应
**解决方案：**
1. 右键点击文件 → "打开方式" → 选择浏览器
2. 或使用本地服务器方案（推荐）
