# 霍兰德职业兴趣测试 - 专业版

基于RIASEC模型的国际权威职业兴趣评估工具。

## 📋 项目简介

霍兰德职业兴趣测试（Holland Career Interest Test）是一个专业的职业兴趣评估工具，采用是/否答题模式，包含120道题目，提供全面的职业分析报告。

## ✨ 主要特性

- **120道专业题目**：每个职业兴趣类型20题，确保评估全面准确
- **是/否答题模式**：简洁明了，易于操作
- **全面分析报告**：包含职业兴趣、职业倾向、职业定位、职业指导等
- **专业报告生成**：详细报告和简要报告两种格式

## 🚀 快速开始

### 安装

无需安装，直接运行即可（Python 3.7+）。

### 运行测试

```bash
python -m holland_test.main
```
