========================================
如何打开霍兰德测试页面 - 最简单方法
========================================

【方法 1：一键启动（推荐）】
1. 双击 "一键启动.bat"
2. 等待几秒钟，浏览器会自动打开
3. 如果浏览器没有自动打开，手动访问：
   http://localhost:8000/holland_test_preview.html

【方法 2：诊断问题】
如果方法1失败，运行 "诊断问题.bat" 查看具体问题

【方法 3：直接打开文件】
1. 找到 holland_test_preview.html 文件
2. 右键点击 → "打开方式" → 选择浏览器
3. 勾选"始终使用此应用打开.html文件"

【方法 4：从浏览器打开】
1. 打开浏览器（Chrome、Edge、Firefox）
2. 按 Ctrl + O（或 文件 → 打开文件）
3. 找到并选择 holland_test_preview.html
4. 点击"打开"

如果所有方法都失败
========================================
1. 运行 "诊断问题.bat" 查看详细错误信息
2. 检查 Python 是否安装（运行: python --version）
3. 检查文件是否在正确位置
4. 尝试使用不同的浏览器
