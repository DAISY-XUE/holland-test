# 快速开始指南

## 🚀 3步快速启动

### 步骤1：安装依赖

```powershell
pip install -r requirements.txt
```

### 步骤2：运行程序

**工作流版本（推荐）**：
```powershell
python -m pdf_auto_renamer.workflow_main
```

**或原有版本**：
```powershell
python -m pdf_auto_renamer.main
```

### 步骤3：放入PDF文件

将PDF文件放入：`C:\Users\你的用户名\Desktop\扫描文件夹`

程序会自动：
- 📄 提取文本
- 🔍 识别类型（合同/发票/审核报告等）
- ✏️ 重命名文件

---

## 📋 支持的文件类型

- ✅ **合同**：自动提取合同编号和项目名称
- ✅ **发票**：识别月份和费用类型
- ✅ **审核报告**：提取项目名称
- ✅ **劳务费相关**：审批材料、付款申请单、发票
- ✅ **收据**：资金往来统一票据等

---

## 🎯 命名示例

| 原文件名 | 识别类型 | 新文件名 |
|---------|---------|---------|
| `扫描001.pdf` | 合同 | `HT-2024-001_XXX工程项目.pdf` |
| `发票.pdf` | 发票 | `5月劳务费发票.pdf` |
| `报告.pdf` | 审核报告 | `(审核报告)XXX工程项目.pdf` |

## ❓ 遇到问题？

1. **检查依赖**：`pip list | findstr prefect`
2. **查看日志**：`扫描文件夹\auto_renamer.log`
3. **检查监控目录**：确保 `扫描文件夹` 目录存在
