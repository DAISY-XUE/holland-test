# 霍兰德职业兴趣测试 - 专业版

霍兰德职业兴趣测试（Holland Career Interest Test），也称为RIASEC模型，是一种广泛使用的职业兴趣评估工具，被国际企业和组织广泛采用。

## 测试特点

### 专业性
- 基于国际公认的霍兰德职业兴趣理论（RIASEC模型）
- 参考标准霍兰德职业兴趣量表设计
- 120道题目，每个类型20题，确保评估全面准确
- 是/否答题模式，简洁明了

### 全面性
- **职业兴趣分析**：识别6种职业兴趣类型及其强度
- **职业倾向分析**：分析工作风格和团队偏好
- **职业定位分析**：明确适合的行业和职业层级
- **职业指导建议**：提供立即行动、技能发展、职业路径等建议

## 六种职业兴趣类型

- **R (Realistic) - 现实型**：喜欢使用工具、机器和设备进行实际操作
- **I (Investigative) - 研究型**：喜欢观察、学习、研究、分析和解决问题
- **A (Artistic) - 艺术型**：喜欢自由表达、创作和艺术活动
- **S (Social) - 社会型**：喜欢帮助他人、教学和照顾他人
- **E (Enterprising) - 企业型**：喜欢领导、影响他人和销售管理
- **C (Conventional) - 常规型**：喜欢数据、记录、组织和遵循程序

## 使用方法

### 方式1：直接运行模块（推荐）

```bash
python -m holland_test.main
```

程序将：
1. 显示测试说明和欢迎信息
2. 依次显示120道题目
3. 收集您的答案（是/否）
4. 自动计算结果并生成专业报告
5. 可选择保存详细报告或简要报告

### 方式2：作为库使用

```python
from holland_test.questions import get_all_questions
from holland_test.scorer import score_test
from holland_test.report_generator import generate_professional_report

# 获取题目
questions = get_all_questions()

# 准备答案（题目ID到布尔值的映射）
answers = {
    1: True,   # 第1题回答"是"
    2: False,  # 第2题回答"否"
    # ... 其他答案
}

# 计算结果
result = score_test(answers)

# 生成专业报告
report = generate_professional_report(result)
print(report)
```

## 答题说明

- **是/否模式**：每道题目只需回答"是"或"否"
- **输入方式**：
  - 输入 `Y`、`y`、`1` 或 `是` 表示"是"
  - 输入 `N`、`n`、`0` 或 `否` 表示"否"
  - 输入 `Q` 或 `q` 可以随时退出测试
- **答题建议**：
  - 请根据您的真实想法和感受回答
  - 不要过度思考，第一反应往往更准确
  - 建议在安静、不受干扰的环境下完成
  - 预计完成时间：15-20分钟

## 报告内容

生成的报告包括：

1. **测试概况**：测试类型、题目数量、答题方式
2. **得分统计**：各类型得分、百分比、强度等级
3. **核心结果**：主要类型、次要类型、第三类型、类型组合
4. **职业兴趣分析**：详细描述主要兴趣类型的特点
5. **职业倾向分析**：工作风格和团队偏好
6. **职业定位分析**：适合的行业领域和职业层级
7. **推荐职业方向**：基于测试结果推荐的职业列表
8. **职业指导建议**：立即行动、技能发展、职业路径、工作环境建议
9. **测试说明**：关于测试结果的解释和使用建议

## 报告保存

测试完成后，可以选择：
- **保存详细报告**：包含完整的分析和建议
- **保存简要报告**：快速查看核心结果和推荐职业

报告文件命名格式：
- 详细报告：`holland_test_result_YYYYMMDD_HHMMSS.txt`
- 简要报告：`holland_test_summary_YYYYMMDD_HHMMSS.txt`

## 注意事项

1. 本测试基于霍兰德职业兴趣理论（RIASEC模型），是国际公认的职业兴趣评估工具
2. 测试结果反映的是职业兴趣倾向，而非能力评估
3. 职业选择应综合考虑兴趣、能力、价值观、市场需求等多方面因素
4. 建议结合其他评估工具（如能力测试、性格测试等）进行综合评估
5. 职业发展是一个动态过程，建议定期重新评估
6. 本报告仅供参考，具体职业选择请结合个人实际情况和专业咨询

## 适用场景

- **个人职业规划**：了解自己的职业兴趣，制定职业发展计划
- **企业招聘**：评估候选人的职业兴趣匹配度
- **教育咨询**：帮助学生选择专业和职业方向
- **转职评估**：分析转职的适合性和方向
- **团队建设**：了解团队成员的工作偏好和协作方式
